## Description

**Cub3D** is a group project that aim to create a 3D graphical representation of the inside
of a maze from a FPP (First-person perspective) using the concept of ray-casting.


## Instructions

Compile the project with :
````
make
````

Launch the program with :
````
./cub3D <filename>
````
